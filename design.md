# KULAMA HairCare — Design System

> Generated by **anydesign** skill • Aesthetic: **"Braided Cathedral — Wrocław hair salon meets editorial fashion magazine"**.
> Visual idea: the page is **split vertically down the middle**. From the **top** of the viewport a **stylized 3D girl** leans down and looks *into* the page, her **long 3D braids cascading and swaying** through the centre seam, draping over both halves of the layout.

## 1. Brand essence

- Owner: salon in Wrocław, Poland — KULAMA Hair Care & Braids
- Specialties: Box Braids, Cornrows, Knotless Braids, Twist Styles
- Voice: warm, expert, editorial. Multi-cultural African heritage meets Polish minimalism.
- Trust markers: 500+ clients · 5+ years · Wrocław Rynek address · @kulama_braids on IG.

## 2. Visual story (the 3D narrative)

```
   ┌──────────────────────────────────────────────────┐
   │              ▲ stylized 3D girl ▲                │
   │             head looks DOWN from top             │
   │  ┌─────────┐ │ braids ── │  ┌──────────────────┐ │
   │  │  LEFT   │ │  swaying  │  │     RIGHT        │ │
   │  │  PANEL  │ │  in 3D    │  │     PANEL        │ │
   │  │ (story) │ │  through  │  │  (booking/CTA)   │ │
   │  │         │ │  centre   │  │                  │ │
   │  └─────────┘ │  seam     │  └──────────────────┘ │
   └──────────────────────────────────────────────────┘
```

- Page bg: warm cream (paper) with subtle grain texture.
- Centre seam: 1–2px hairline of deep brown (#2A1810).
- Braids: 8–12 procedural ropes hanging from the top, animated by sin + perlin offsets, longer than the viewport so they always fall past the fold.
- Head: low-poly stylized "looking down" silhouette — a soft dome with hint of cheek + jawline + hair line, very abstract, never aiming for a literal portrait.

## 5. Layout system — the split-screen

- Viewport split at exactly 50% (centre seam). On `lg+` only — mobile collapses to single column under the head.
- Both panels share a 12-col grid with 24px gutter, but render independently.
- `min-height: 100vh` on hero. Subsequent sections may break out (`grid-cols-1 lg:grid-cols-2`).
- Centre lane: a `pointer-events-none` overlay (`fixed inset-0 -z-0`) that hosts the R3F Canvas with the head and braids. The braids cross over both halves naturally because they are 3D.

## 7. Motion language

- **Section reveal**: `opacity 0 → 1`, `y +24 → 0`, `0.7s` `easeOut`.
- **Braids**: continuous procedural sway (`useFrame`); occasional gust gesture every ~6s adds a stronger lateral push.
- **Head**: 1.5deg yaw + 0.8deg pitch oscillation, very slow.
- **Hover on tile/card**: lift `-3px`, soft shadow.
- **Lenis**: `lerp 0.08`, smooth wheel.
- **Reduced motion**: braids freeze in their resting wave, head still, Lenis disabled.

## 8. Accessibility

- All text contrast verified ≥ 4.5:1 (`ink-900` on `bg-paper` ≈ 11.6:1).
- 3D scene is decorative → `aria-hidden`, `pointer-events-none`.
- `prefers-reduced-motion` respected on all animations.
- Keyboard focus rings: 2px `brand-amber`, offset 2.
